**Conceitos Básicos de HTML e CSS**

HTML (HyperText Markup Language) e CSS (Cascading Style Sheets) formam a base da maioria dos sites na internet. Esses conceitos básicos ajudarão você a entender como eles funcionam.

**Quando e por quem essas linguagens foram criadas?**

* **HTML:** Tim Berners-Lee, 1991
* **CSS:** Håkon Wium Lie, 1994

**HTML: Estrutura e Conteúdo**

O HTML é uma linguagem de marcação que define a estrutura e o conteúdo de um site. Ele usa tags para indicar o tipo de conteúdo, como cabeçalhos, parágrafos e links.

**Exemplo de tags para iniciantes:**

* `<html>`: Define o documento HTML
* `<head>`: Contém informações sobre o documento, como título e metadados
* `<body>`: Contém o conteúdo visível do site
* `<h1>`: Cabeçalho de primeiro nível
* `<p>`: Parágrafo
* `<a>`: Link

**CSS: Estilo e Apresentação**

O CSS é uma linguagem de estilo que controla a aparência do conteúdo HTML. Ele usa regras para definir cores, fontes, margens e outros atributos visuais.

**Exemplo de regras CSS:**

* `h1 { color: red; }`: Torna todos os cabeçalhos de primeiro nível vermelhos
* `p { font-size: 12px; }`: Define o tamanho da fonte para todos os parágrafos como 12 pixels

**Conceitos Básicos Adicionais**

* **Seletores:** Usados para selecionar elementos HTML aos quais aplicar regras CSS
* **Cascata:** O CSS segue uma ordem de cascata, com regras posteriores sobrescrevendo as anteriores
* **Herança:** Os elementos HTML herdam os estilos de seus elementos pais
* **Estilo embutido:** Estilos podem ser adicionados diretamente ao código HTML usando atributos de estilo
* **Estilo no cabeçalho:** Estilos podem ser definidos na seção `<head>` de um documento HTML, aplicando-se a todo o documento
* **Estilo externo:** Estilos podem ser armazenados em um arquivo CSS externo e vinculados a um documento HTML

**Conclusão**

Compreender os conceitos básicos de HTML e CSS é essencial para criar sites funcionais e visualmente atraentes. Essas linguagens fornecem a estrutura e o estilo necessários para entregar conteúdo da web de forma eficaz.
